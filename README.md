# RA Privacy Policy

This repository hosts the privacy policy for RA – Site Machine Management
(macOS / Windows desktop app, bundle ID com.armada.rapc).

Published site (served by GitHub Pages):

- 中文：https://henry8055.github.io/ra-privacy/zh/
- English: https://henry8055.github.io/ra-privacy/en/

The URL above is referenced by App Store Connect → App Information → Privacy Policy URL.

## How to update

    git clone https://github.com/Henry8055/ra-privacy.git
    cd ra-privacy
    # edit zh/index.md and en/index.md, bump the Last Updated date
    git commit -am "chore: update privacy policy"
    git push

GitHub Pages will redeploy within 1-3 minutes. App Store Connect needs no change.
